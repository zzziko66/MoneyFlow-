# MoneyFlow · 个人财务管理系统

> 一款纯前端个人财务记账 SPA — 无需后端，开箱即用。

MoneyFlow 帮助你轻松追踪每一笔收支，通过可视化图表分析消费结构，借助 AI 获得理财建议，以简约精致的方式掌握自己的财务状况。

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 结构 | **HTML5** | 语义化标签，无障碍属性 |
| 样式 | **CSS3** | 自定义属性主题系统、Grid / Flexbox 布局、3D transform 动画、Canvas 2D 图表渲染 |
| 逻辑 | **JavaScript ES6+** | IIFE 模块化、箭头函数、解构赋值、async/await 异步处理 |
| 存储 | **localStorage** | 多用户数据隔离，按邮箱键值独立存储 |
| 图表 | **Canvas 2D API** | 自绘条形图 + 环形图，不依赖任何图表库 |
| 导出 | **html2canvas** | 账单小票导出 PNG（按需动态加载，不阻塞页面） |

## 页面功能详解

### 登录 / 注册（index.html）

用户身份管理入口，支持新用户注册和老用户登录。

**注册流程（四步向导）：**

| 步骤 | 内容 | 说明 |
|------|------|------|
| ① 资料 | 昵称 + 电子邮箱 | 实时邮箱格式校验，输入框聚焦时自动提示 |
| ② 账户 | 选择支付方式 | 微信 / 支付宝 / 银行卡 / 现金，可多选，默认全选 |
| ③ 预算 | 月预算上限 + 预警阈值 | 默认 ¥3000，阈值可选 60% / 80% / 90% |
| ④ 安全 | 设置密码 + 同意协议 | 实时密码强度指示器（弱/中等/强/非常强），勾选后方可提交 |

**登录功能：**
- 邮箱 + 密码验证登录
- 登录 / 注册表单平滑切换动画
- **测试账户一键登录**：点击「🧪 测试账户快速登录」自动填入预置账户，一键进入

> 已登录用户访问此页会自动跳转到财富视图。

### 财富视图（dashboard.html）

仪表盘是应用的核心总览页，登录后默认进入。页面分为三个区域：

#### 1. 三宫格数据卡片

| 卡片 | 功能 | 交互细节 |
|------|------|----------|
| **净资产** | 总收入 − 总支出 | 深色渐变背景，脉冲光晕动画，金额数字跳动入场 |
| **本月支出** | 当月累计支出 | 显示支出总额及当月收入对比，支出用深色、收入用绿色区分 |
| **预算进度** | 预算使用率可视化 | 水位条动画填充，预警线标记阈值位置，超阈值变黄、超支变红 |

每张卡片均支持 **3D 倾斜交互**：鼠标悬停时卡片跟随光标产生细腻的三维旋转（X/Y 轴 ±15°），同时表面高光（glare）随鼠标位置动态移动，移出时平滑复位。

#### 2. 消费结构分析

位于页面下半部分，左右分栏展示：

- **条形图**（左侧，占 3/5 宽度）：横向展示各支出分类，类别名称（含 emoji 图标）居左，百分比显示在色块内部（白色文字），具体金额紧随色块右侧，比例一目了然。
- **环形图**（右侧，占 2/5 宽度）：彩色扇区展示占比，白色描边分隔，中心镂空显示总支出金额。

> 两张图的配色自动跟随当前主题变化（5 套主题对应 5 套色板），切换主题时图表即时重绘。

#### 3. 智能诊断

根据当月收支数据自动生成一句简短财务建议：
- 预算超支时预警
- 支出环比增长时提醒
- 餐饮占比过高时给出建议
- 正常状态时给予肯定

#### 4. 快速记账弹窗

点击「＋记一笔」按钮打开弹窗，支持：
- 支出 / 收入类型切换
- 分类选择（支出：餐饮/交通/购物/娱乐/居住/医疗；收入：工资/其他）
- 事项名称（选填，默认使用分类名）
- 金额输入
- 账户选择（关联用户在注册时勾选的账户）
- 提交后即时刷新所有统计数据和图表，并弹出 Toast 反馈

### 账单（transactions.html）

收支记录的集中查看与管理页面。

#### 数据总览

顶部汇总卡片显示：
- **总收入** — 所有收入记录合计（绿色）
- **总支出** — 所有支出记录合计（深色）
- **总笔数** — 记录总数

三个数字均带有入场数字跳动动画。

#### 筛选与列表

- **过滤药丸按钮**：全部 / 支出 / 收入三档快速切换
- **交易条目**：每条记录展示分类图标、标题、分类名称、支付账户、时间、金额
  - 支出金额为深色（带负号），收入金额为绿色（带正号）
  - 鼠标悬停时条目轻微右移并浮现阴影

#### 今日小票

点击「🖨️ 打印今日账单」打开小票弹窗：

- **小票样式**：模拟热敏打印机出纸效果，Courier New 等宽字体，黑白配色，顶部和底部有锯齿边缘
- **内容**：钱流记品牌标题 → 日期 → 分隔虚线 → 今日每笔收支明细 → 分隔虚线 → 收入/支出/净额汇总 → "made by 钉子" 页脚
- **下载**：点击「⬇ 下载小票」后动态加载 html2canvas 库（仅首次需要），将小票区域渲染为高清 PNG 图片并自动下载
- 今日无记录时显示空状态提示

### AI 顾问（insights.html）

智能财务分析 + 对话系统，双模式运行。

#### AI 财务分析

- 自动汇总当前用户的收支数据（本月收入/支出/净资产/分类占比/预算使用率）
- 调用 AI 生成个性化理财建议（120 字以内），并以**打字机效果**逐字输出
- 支持刷新重新生成
- 无数据时提示用户先去记账

#### AI 对话窗口

与财务助手进行自然语言交流，支持的话题：

| 话题 | 触发词示例 | 回复内容 |
|------|-----------|----------|
| 支出分析 | "我花了多少钱"、"支出" | 列出总支出、最大支出类别及占比、预算使用率 |
| 收入情况 | "收入"、"工资"、"赚" | 收入金额、支出对比、储蓄率及评价 |
| 净资产 | "净资产"、"总资产"、"身价" | 净资产数值及状态评估 |
| 预算 | "预算"、"超了"、"预警" | 预算使用详情及温馨提示 |
| 分类明细 | "餐饮"、"交通"、"购物" | 该分类支出金额、占比及针对性建议 |
| 省钱建议 | "怎么省"、"建议"、"优化" | 基于数据分析给出可操作建议 |
| 基础问候 | "你好"、"嗨"、"在吗" | 带时间感知的问候（早上/下午/晚上） |
| 兜底回复 | 其他问题 | 展示财务概览并引导提问 |

**双模式机制：**
- **Mock 模式**（默认）：无需任何 API 密钥，通过关键词匹配 + 真实财务数据生成回复，完全离线可用
- **DeepSeek 模式**：在设置弹窗中填入 DeepSeek API 密钥后自动切换，调用大模型生成更智能的回答；API 不可用时自动降级为 mock 模式

## 侧边栏与全局功能

所有页面（除登录/注册外）共享同一侧边栏组件：

- **导航**：财富视图 / 账单 / AI 顾问，当前页面高亮标记
- **用户菜单**：点击用户区域展开下拉菜单
  - ⚙️ 修改账户 — 编辑昵称、预算上限、预警阈值、支付方式
  - 🎨 切换主题 — 弹出主题选择窗口
  - ⏻ 退出登录 — 清除会话，返回登录页
- **实时时钟**：页面右上角显示当前时间（时:分:秒），顶部显示问候语

## 数据存储

所有数据使用浏览器 `localStorage` 存储，无需后端服务器：

| Key | 内容 |
|-----|------|
| `moneyflow_users` | 注册用户列表 `{ email: { password, nickname } }` |
| `moneyflow_session` | 当前登录会话（邮箱） |
| `moneyflow_data_{email}` | 每个用户的独立数据：用户信息、账户列表、交易记录数组 |
| `mf_theme` | 用户主题偏好 |
| `moneyflow_ai_config` | AI API 配置（API Key / 模型选择） |
| `mf_copy_*` | AI 品牌文案缓存 |

> 清除浏览器数据会导致所有记录丢失，请定期备份。

## 本地运行

无需安装任何依赖，无需构建工具，直接用浏览器打开 `index.html` 即可：

```bash

open index.html     # macOS
```

## 项目特点

- **零依赖** — 纯 HTML / CSS / JavaScript，不依赖任何框架或第三方库（html2canvas 按需动态加载）
- **多用户系统** — 完整注册/登录/登出流程，数据按邮箱隔离
- **纯前端持久化** — localStorage 存储，无需后端服务器
- **5 套主题** — CSS 自定义属性驱动的完整皮肤系统，图表颜色同步适配
- **3D 交互** — 卡片级三维倾斜 + 动态高光，提升操作沉浸感
- **自绘图表** — Canvas 2D API 自实现条形图与环形图，不依赖第三方图表库
- **AI 双模式** — 支持离线 mock + DeepSeek API 双模式智能分析
- **小票导出** — 热敏打印机风格小票，支持高清 PNG 下载
- **响应式设计** — 适配桌面与移动端，移动端底部导航栏优化
